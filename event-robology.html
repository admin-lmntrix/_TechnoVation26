Robology

Robotics Event

Design, build and program a robot capable of solving a set task, then run it on the arena.

TrackOff-Stage
Team3 per team
Classes9–12
Rounds2

The Brief

Design and build a robot to solve a set task. You'll build and program a basic robot to complete a predefined challenge.

Format

  • Round 1 — design and build.
  • Round 2 — performance testing on the arena.

Judging

  • Innovation, precision and problem-solving.
  • Functionality, efficiency and design of the build.

All participants report at 7:30 AM in proper school uniform, with a valid School ID and a Teacher-in-Charge. Read the full guidelines →
